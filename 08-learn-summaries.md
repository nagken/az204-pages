# Microsoft Learn Path Summaries

> Curated path that maps to the AZ-204 outline. Use as a structured study route.

## 1. Develop Azure compute solutions

- [**Implement Azure App Service web apps**](https://learn.microsoft.com/training/paths/create-azure-app-service-web-apps/) - App Service basics, plans, deployment, configuration, scaling, slots.
- [**Implement Azure Functions**](https://learn.microsoft.com/training/paths/implement-azure-functions/) - triggers, bindings, hosting plans, Durable Functions.
- [**Implement containerized solutions**](https://learn.microsoft.com/training/paths/az-204-implement-iaas-solutions/) - ACR, ACI, Container Apps.

## 2. Develop for Azure storage

- [**Develop solutions that use Azure Cosmos DB**](https://learn.microsoft.com/training/paths/az-204-develop-solutions-that-use-azure-cosmos-db/) - APIs, partitioning, consistency, change feed.
- [**Develop solutions that use Azure Blob Storage**](https://learn.microsoft.com/training/paths/develop-solutions-that-use-blob-storage/) - operations, lifecycle, properties + metadata.

## 3. Implement Azure security

- [**Implement user authentication and authorization**](https://learn.microsoft.com/training/paths/az-204-implement-authentication-authorization/) - Microsoft Identity Platform, MSAL, OAuth flows, Microsoft Graph.
- [**Implement secure Azure solutions**](https://learn.microsoft.com/training/paths/az-204-implement-secure-cloud-solutions/) - Key Vault, App Configuration, Managed Identity.

## 4. Monitor, troubleshoot, and optimize Azure solutions

- [**Instrument solutions to support monitoring and logging**](https://learn.microsoft.com/training/paths/az-204-instrument-solutions-support-monitoring-logging/) - Application Insights instrumentation, sampling, alerts, KQL.

## 5. Connect to and consume Azure services

- [**Implement API Management**](https://learn.microsoft.com/training/modules/explore-api-management/) - APIM tiers, products, policies.
- [**Develop event-based solutions**](https://learn.microsoft.com/training/paths/az-204-develop-event-based-solutions/) - Event Grid, Event Hubs.
- [**Develop message-based solutions**](https://learn.microsoft.com/training/paths/az-204-develop-message-based-solutions/) - Service Bus, Storage Queue.

## Suggested 30-day route

| 2 | Storage + Security | 02, 03 |
