# Glossary and Acronyms

> Common AZ-204 terms. Pair with **Concept & Reference Index**.

## A

- **ACR** - Azure Container Registry. Private OCI image registry.
- **ACI** - Azure Container Instances. Single container or container group, no orchestrator.
- **ACA** - Azure Container Apps. Managed container platform with KEDA + Dapr.
- **APIM** - Azure API Management. Front door for backend APIs.
- **App Service Plan** - Pricing tier and compute resource shared by App Service apps.
- **App Configuration** - Centralized config + feature flags + Key Vault references.
- **Application Insights (AI)** - APM module of Azure Monitor.

## B

- **Blob** - Binary Large Object. Block / Append / Page types.
- **Bindings** - Functions input/output declarations replacing SDK code.

## C

- **Change Feed** - Persistent log of Cosmos inserts and updates.
- **CloudEvents** - Open spec for event payloads (v1.0).
- **Cold start** - Delay to spin up an idle Functions Consumption instance.
- **Consumption Plan** - Functions hosting that scales-to-zero, pay-per-execution.
- **Cosmos DB** - Globally distributed multi-model NoSQL database.
- **CRUD** - Create, Read, Update, Delete.

## D

- **Dapr** - Distributed Application Runtime sidecar in Container Apps.
- **DLQ** - Dead-letter queue (Service Bus / Event Grid).
- **Durable Functions** - Stateful workflow extension on Functions.

## E

- **Entra ID** - New name for Azure Active Directory.
- **Event Grid** - Reactive event broker.
- **Event Hubs** - High-throughput streaming platform (Kafka-compatible).

## F

- **Federated Credential** - Workload identity federation (no client secret).
- **Flex Consumption** - Newer Functions hosting plan, VNet + scale-to-zero.

## G

- **Graph API** - Microsoft Graph: unified REST endpoint for Microsoft 365 + Entra data.

## I

- **Isolated Worker** - Out-of-process Functions execution model in .NET.

## K

- **KEDA** - Kubernetes Event-Driven Autoscaling. Container Apps scale engine.
- **Key Vault** - Service for secrets, keys, and certificates.
- **KQL** - Kusto Query Language used by Azure Monitor / Application Insights.

## M

- **Managed Identity (MI)** - Azure-managed identity for a resource. System or User assigned.
- **Microsoft Graph** - REST API for Microsoft 365 / Entra ID.
- **MSAL** - Microsoft Authentication Library.

## O

- **OAuth 2.0** - Authorization framework underlying Microsoft Identity Platform.
- **OBO** - On-Behalf-Of flow.
- **OpenTelemetry (OTel)** - Vendor-neutral telemetry standard. Azure Monitor distro available.

## P

- **PKCE** - Proof Key for Code Exchange. Used in public clients.
- **PaaS / IaaS / FaaS / SaaS** - Platform / Infra / Function / Software as a Service.
- **Partition Key** - Cosmos DB shard key.

## R

- **RBAC** - Role-Based Access Control.
- **RU/s** - Request Units per second. Cosmos DB throughput unit.
- **Revision** - Immutable Container Apps version.

## S

- **SAS** - Shared Access Signature. Account / Service / User-delegation.
- **Service Bus** - Enterprise pub/sub + queue messaging.
- **Service Connector** - Azure feature to wire managed identities + connection strings between resources.
- **Slot** - App Service deployment target with swap.

## T

- **TelemetryClient** - Application Insights SDK client.
- **TLS** - Transport Layer Security.
- **TTL** - Time To Live.

## U

- **User Delegation SAS** - SAS signed with an Entra ID OAuth token (no shared key).

## V

- **VNet** - Azure Virtual Network.
- **VNet Integration** - App Service / Functions outbound connectivity to a VNet.
